### 🔑 Key Features:
- Real-time monitoring of drug stock levels  
- Expiry date tracking  
- Supply chain tracking (warehouse → hospital)  
- Automated alerts for low stock and expiry  

The system uses a **centralized database** and provides **role-based access control** for different users.
